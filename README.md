# LockIT

LockIT is an open-source remote device management platform designed for trusted devices and trusted users.

The project enables secure, permission-based remote actions between paired Android devices through a modern Android application, web dashboard, and backend infrastructure.

Whether it's locking a phone that was accidentally left awake, checking battery status, running automations, or managing multiple trusted devices, LockIT provides a secure and privacy-focused solution built around explicit user consent.

## Core Features

- 🔒 Remote screen lock
- 📱 Live device monitoring
- 🔋 Battery and charging information
- ⚡ Automation and scheduled actions
- 🌐 Web dashboard
- 📲 Multi-device management
- 🔐 Secure pairing and authentication
- 📡 Real-time communication
- 🔄 GitHub release update checking
- 🧩 Modular architecture for future expansion

## Principles

- Privacy First
- Explicit User Consent
- Secure by Default
- Lightweight & Fast
- Extensible Architecture

LockIT is intended for personal devices, family members, and trusted relationships—not for unauthorized monitoring or surveillance.
